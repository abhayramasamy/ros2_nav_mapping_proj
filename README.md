# Autonomous Navigation with ROS 2 — Nav2

A small exploratory robotics project completed as part of the ROBO AI 45-day hands-on robotics workshop.

## What this does
Implements mapping and goal based autonomous navigation using the ROS 2 Nav2 stack. The robot is manually commanded to explore and map the environment using SLAM, then navigates autonomously to goal positions once the map is built.

## Stack used
- ROS 2 Jazzy
- Nav2
- Gazebo simulation
- SLAM Toolbox
- RViz2

## What was learned
- Setting up a full ROS 2 navigation pipeline from scratch
- Manual SLAM based mapping
- Goal based autonomous navigation
- ROS 2 node architecture and topic communication
- Simulation in Gazebo

## Context
Built during the ROBO AI robotics workshop where core robotics concepts including IoT, hardware communication protocols (SPI, I2C, UART), PCB design basics, inverse kinematics and AI fundamentals were covered over 45 days.

## Status
Exploratory project. Navigation pipeline functional in simulation. Code reflects learning stage, not production level implementation.
